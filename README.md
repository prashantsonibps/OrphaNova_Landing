## OrphaNova Landing (Labs Marketing Site)

This is the marketing landing page for **OrphaNova Labs**, built with **Vite + React** and a dark-only theme.
The page explains the mission, product, and pricing, and links to the live Labs app at `https://lab.orphanova.com/`.
The **Sign in** button and CTAs open the existing Labs app and Notion calendar for demos.

### Run locally

```bash
npm install
npm run dev
```

### Build for production (Vercel / static hosting)

```bash
npm run build
```
